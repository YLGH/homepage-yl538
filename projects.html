PyLand

Pyland aims to provide a fun and creative environment on the Raspberry Pi (or other platforms) to aid children learning programming and Computer Science Concepts in Python. It was started as a summer intern project at the University of Cambridge Computer Laboratory June-August 2014 supported by a grant from the Broadcom Foundation. Check out the Github page here.

SplitBit

SplitBit was a project created during my second year of university for Frontier, a games company based in Cambridge. SplitBit is an online equity and salary negotiation tool for startup founders in the tech industry. It used Node.js for a front end and Java for an artificial intelligent backend that "guessed" how much equity/salary a person should should given his qualifications.

Ray Tracer

I wrote a ray tracer in Java for my graphics course during my second year of university. It can draw spheres, rectangles, planes and support multiple light soruces with different anti-aliasing and sampling methods. Check out the Github page here.
